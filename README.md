# ZBuffer

This is a software [Z-Buffer](http://en.wikipedia.org/wiki/Z-buffer) implementation.

## How to build

```
$ qmake
$ make
```

Successfully built on Manjaro Linux with Qt 5.

## How to run

```
$ ./zbuffer bunny.obj
$ ./zbuffer budda.obj
$ ./zbuffer dragon.obj
```
